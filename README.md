# ooplab3
# Lab 3 — Matrix & Vector System (OOP in C#)

## Description
This project implements a Matrix and Row system using Object-Oriented Programming concepts in C#.

## OOP Concepts Applied
- Encapsulation
- Composition
- Aggregation
- Dependency

## Classes

### Row Class
- Stores values in a double array
- Provides getter and setter methods

### Matrix Class
- Contains multiple Row objects
- Supports matrix operations

## How to Run
1. Open the .sln file in Visual Studio
2. Build the solution
3. Run the program
